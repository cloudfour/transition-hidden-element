Transition Hidden Element Demo

A demo showing this library in action with different setting configured.

Default Settings

Fade an element in and out 

      
const simpleFader = transitionHiddenElement({
  element: document.querySelector('.js-simple-fade'),
  visibleClass: 'is-shown'
});

I should fade in and out. After fading out, hidden should be applied to me.

Fade In But Not Out

      
const fadeIn = transitionHiddenElement({
  element: document.querySelector('.js-fade-in'),
  visibleClass: 'is-shown',
  waitMode: 'immediate'
});

I should fade in but not out.

Fade Out With a Timeout

      
const fadeOutTimeout = transitionHiddenElement({
  element: document.querySelector('.js-fade-out-timeout'),
  visibleClass: 'is-shown',
  waitMode: 'timeout',
  timeoutDuration: 300
});

I should fade in and out. After fading out, hidden should be applied to me.

Fade In and Out with `display: none;`

      
const fadeInOutDisplay = transitionHiddenElement({
  element: document.querySelector('.js-fade-in-out-display'),
  visibleClass: 'is-shown',
  hideMode: 'display',
  displayValue: 'block'
});

I should fade in and out. After fading out, display: none; should be applied to me.
